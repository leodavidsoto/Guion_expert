# TEMPLATES TIKTOK

## 1. STORYTIME
```
[0-1s] "STORYTIME: [Título clickbait pero real]"
[1-3s] "Okay so..."
[3-60s] Narrativa directa a cámara
- Expresiones faciales exageradas
- Manos gestuales
- Cuts cada 3-5 palabras
[60s+] Plot twist o conclusión
Comments: "Part 2?", "No way", etc.
```

## 2. POV (Point of View)
```
[0-2s] "POV: [Situación relatable]"
[2-30s] Actuación de la situación
- Puede ser humor o drama
- Reacciones exageradas
- Audio trending que match
[30-60s] Punchline o plot twist
Duet/Stitch bait: Dejar abierto a respuestas
```

## 3. PARTE 1, 2, 3... (SERIES)
```
[0-2s] "PARTE [#]: [Título de serie]"
[2-5s] "Recap:" [Si no es parte 1]
[5-180s] Contenido de este capítulo
- Cliffhanger al final
- "¿Qué creen que pasó?"
[180s] "PARTE [#+1] MAÑANA"
Strategy: Subir partes en días consecutivos
```

## 4. CHALLENGE/TREND
```
[0-1s] Inicio del trend (audio específico)
[1-30s] Tu twist único del challenge
- Mantén estructura del trend
- Añade personalidad/contexto único
[30-60s] Cierre con tu toque
Caption: #[TrendName] + "My version"
```

## 5. TUTORIAL RÁPIDO
```
[0-2s] "Cómo [HACER ALGO] que nadie te enseñó"
[2-10s] "Esto me hubiera ahorrado [TIEMPO/DINERO]"
[10-60s] Pasos ultra rápidos
- Un cut por paso
- Captions grandes en cada paso
[60s] "Funciona, confía"
```

## 6. REACTION/DUET
```
[0-3s] Tu reacción inicial (shock/risa/wtf)
[3-30s] Video original plays
[30-60s] Tu comentario/análisis
- A cámara, directo
- "Wait, but...", "Okay so...", "HOLD ON"
[60s] Conclusión o pregunta al creator original
```

## 7. DAY IN THE LIFE (TIKTOK STYLE)
```
[0-2s] "DITL como [ROL INTERESANTE]"
[2-5s] "Es las [HORA]am y..."
[5-120s] Clips rápidos con timestamps
- "8:30am: [actividad]" (3 seg de clip)
- "10:00am: [actividad]" (3 seg)
- Jumpcuts agresivos
[120-180s] "Y así fue mi día, honestamente..."
```

## 8. GREENSCREEN COMMENTARY
```
[0-2s] Aparece screenshot/video de fondo
[2-5s] "Okay, necesitamos hablar de esto"
[5-60s] Tu análisis/opinión
- Señalas partes del background
- Expresiones a cámara
- Cuts frecuentes
[60s] Hot take final
```

## 9. RELATABLE HUMOR
```
[0-1s] "No one:" o "Me:"
[1-30s] Actuación de situación universal
- Expresiones exageradas
- Audio trending gracioso
[30-60s] Punchline
Comments: "THIS IS SO ME", "WHY IS THIS ACCURATE"
```

## 10. EDUCATIONAL
```
[0-2s] "Algo que aprendí hoy sobre [TEMA]"
[2-10s] Hook con el dato más impactante
[10-50s] Explicación con visuales
- Texto animado
- Gráficos simples
- Analogías visuales
[50-60s] "Sígueme para más [NICHO]"
```
